Overview

Welcome to a monitoring tool developed by hkjn.

The purpose is to learn more about the inner workings of Bitcoin and Lightning Network by making it easier to operate these nodes, and easier to observe what they're doing.

Available consoles:

  • System: System-level metrics
  • lnmon: Monitoring of Lightning Network nodes
  • lnmon_process: Detailed process information for LN nodes
  • bcmon: Monitoring of Bitcoin nodes
